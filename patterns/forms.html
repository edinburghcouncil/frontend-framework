--- title: Forms ---

Structure

In a standard form, all fields should appear in a <li> inside a <ul> inside the <form>.

HTML

<form>
    <ul>
        <li>
            [form field]
        </li>
        <li>
            [form field]
        </li>
        <li>
            [form field]
        </li>
    </ul>
    <button class="button">Submit</button>
</form>

Examples

HTML

<li>
    <label for="text-field">Text</label>
    <input type="text" name="text-field" id="text-field" />
</li>

HTML
